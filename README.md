# Toastmaster Timer Tools (头马时间官助手)

一个专为 Toastmasters（头马国际演讲会）会议设计的专业计时工具。帮助时间官轻松管理会议流程，确保演讲者精准掌握时间。

![Logo](public/logo.png)

![Cover](public/cover.jpg)

## ✨ 主要功能

- **多场景预设**：内置标准的 Toastmasters 计时规则，包括：
  - 即兴演讲 (Table Topics): 1-2 分钟
  - 备稿演讲 (Prepared Speech): 5-7 分钟
  - 点评/评估 (Evaluation): 2-3 分钟
  - 自我介绍 (Self Intro): 30 秒
- **自定义组合**：支持创建和保存自定义的时间组合，满足特殊会议需求。
- **直观的视觉提示**：
  - 🟢 绿牌：达到合格时间
  - 🟡 黄牌：提示时间
  - 🔴 红牌：警告时间
- **声音提醒**：可配置的铃声提醒功能。
- **时间线记录**：
  - 自动记录每位演讲者的具体时长。
  - 自动计算并标记超时/合格状态。
  - 支持添加备注和修改记录。
  - 历史记录查询与筛选。
- **响应式设计**：适配各种屏幕尺寸，侧边栏可折叠，专注计时界面。

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **构建工具**: Vite
- **样式库**: Tailwind CSS
- **状态管理**: React Context + useReducer
- **持久化**: LocalStorage



### 构建生产版本


## ☁️ 部署指南

### 部署到 Zeabur

1. 访问 [Zeabur](https://zeabur.com) 并登录。
2. 点击控制台中的 **Create Project** (创建项目)。
3. 点击 **Create Service** (创建服务) -> 选择 **Git**。
4. 搜索并选择仓库 `toastmaster_timer_tools`。
5. Zeabur 会自动识别 React/Vite 项目并开始部署。
6. 部署完成后，在 **Networking** (网络) 标签页绑定一个域名即可访问。

## 📝 使用指南

1. **选择计时组合**：在左侧侧边栏选择预设的计时模式（如"备稿演讲"）或创建新模式。
2. **开始计时**：输入演讲者姓名（可选），点击"开始计时"。
3. **监控进度**：
   - 界面背景色会随时间变化（绿 -> 黄 -> 红）。
   - 进度条实时显示当前阶段。
4. **结束与记录**：点击"结束"按钮，记录将自动保存到"时间线"页面。
5. **查看历史**：切换到"时间线"页面查看所有历史记录，支持按日期筛选。
