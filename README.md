# Toastmaster Timer Tools (头马时间官助手)

专为 Toastmasters 国际演讲会设计的专业计时与 AI 报告工具。三栏布局，侧边栏管理计时组合，中间栏配置计时会话，右侧全屏计时显示+AI 报告。

## ✨ 主要功能

### ⏱️ 计时核心
- **10 种预设计时组合**：即兴演讲、备稿演讲、评估、自我介绍、角色介绍、主席开场、总主持人、嘉宾分享、即兴主持人、工作坊
- **三段式视觉提示**：🟢 正常时间 → 🟡 橙色时间 → 🔴 警告时间（30s，不计入时间范围）
- **智能拆分规则**：按上限自动拆分 3 段；上限 ≥ 3min 时黄段 60s，否则 30s
- **自定义组合**：分钟+秒分别编辑，支持颜色/铃声/显示配置
- **铃声称预加载**：M4A 压缩格式，Audio 实例预加载，0 延迟播放

### 🤖 AI 时间官报告
- **DeepSeek V4 API** 直连，API Key 浏览器本地存储
- **中英文双语报告**：一次生成同时输出中文和英文报告
- **Markdown 渲染展示**，支持折叠/展开，中英文切换
- **人工编辑模式**：编辑中可对照时间线数据，支持在 AI 版本和人工版本间切换
- **重新生成确认**：已有报告时弹窗确认避免误覆盖
- **导出 PDF**：精美排版，Markdown → HTML 渲染，含计时表格和报告

### 📊 时间线
- **升序/降序切换**：按时间正序/倒序排列
- **行内编辑**：名称、备注可修改，支持软删除/还原
- **测试数据自动注入**：index.html 预置 20 条会议记录

## 🛠️ 技术栈

- **前端框架**: React 19 + TypeScript
- **构建工具**: Vite 3
- **样式库**: Tailwind CSS 3
- **状态管理**: React Context + useReducer
- **持久化**: LocalStorage
- **AI 模型**: DeepSeek V4 Pro

## 🚀 快速开始

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # 生产构建 → dist/
```

## 🤖 AI 报告使用指南

1. 进入「AI 辅助」tab → 填入 DeepSeek API Key → 展开提示词模板可自定义
2. 保存设置后回到「时间线」tab，选择日期
3. 点击第 2 栏「🤖 AI 生成报告」→ 等待生成 → 查看报告
4. 可点击「✏️ 人工编辑」修改报告，「🖨️ 导出 PDF」打印
